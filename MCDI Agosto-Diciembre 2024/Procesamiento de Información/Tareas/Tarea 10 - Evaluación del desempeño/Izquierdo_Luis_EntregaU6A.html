Introduccin

Los modelos de clasificacin se utilizan mucho en el mbito del aprendizaje automtico para resolver problemas que implican la categorizacin de datos en diferentes clases. En este proyecto se toman un conjunto de datos conformado por tuits y se evala el desempeo de distintos modelos de clasificacin (Naive Bayes, SVM, Random Forest y Regresin Logstica) por medio de mtricas como lo son la precisin, recall y medida-F (F1 Score), ya que estas pueden equilibrar la evaluacin entre casos positivos y negativos, sobretodo en conjuntos de datos desbalanceados.

De igual manera, se pretende analizar matrices de confusin para interpretar los patrones de clasificacin (correctos e incorrectos) de cada modelo para identificar cul es el modelo ms adecuado para este experimento.

Preprocesamiento de Datos

Se crea una funcin que:

  • Elimina caracteres especiales y nmeros
  • Convierte el texto a minsculas
  • Elimina stopwords en espaol
  • Lematiza el texto
In[]:
from nltk.corpus import stopwords

# Descargar recursos necesarios de nltk
#nltk.download('stopwords')

# Obtener las stopwords en espaol
stop_words = set(stopwords.words('spanish'))

def limpiar_texto(texto):
    # Eliminar caracteres especiales y nmeros
    texto = re.sub(r'[^a-zA-Z\s]', '', texto)
    # Convertir a minsculas
    texto = texto.lower()
    
    # Eliminar stopwords
    palabras = texto.split()
    palabras = [palabra for palabra in palabras if palabra not in stop_words]
    
    # Lematizar el texto
    palabras_lematizadas = [lemmatizer.lemmatize(palabra, wordnet.VERB) for palabra in palabras]
    texto = ' '.join(palabras_lematizadas)
    
    return texto

# Inicializar el lematizador
lemmatizer = WordNetLemmatizer()

# Aplicar la funcin de limpieza al conjunto de entrenamiento y prueba
train_data['texto_limpio'] = train_data['text'].apply(limpiar_texto)
test_data['texto_limpio'] = test_data['text'].apply(limpiar_texto)

Entrenar Modelos

Se entrenan con los datos de entrenamiento (X_train y y_train) los sigientes modelos:

  • Naive Bayes (Macro y Micro)
  • SVM (Macro y Micro)
  • Random Forest (Macro y Micro)
  • Regresin Logstica (Macro y Micro)

Macro: Calcula el promedio de la mtrica para cada clase y luego promedia esos resultados. Es til cuando todas las clases son igualmente importantes.

Micro: Calcula la mtrica global considerando todas las predicciones. Es til cuando hay un desbalance de clases.

De igual manera, se imprimen las mtricas para ver el desempeo de cada modelo.
