Inciso 1

La principal similitud entre los mtodos de Biseccin, de Newton y de la secante es que son mtodos iterativos, al igual que los 3 se basan en encontrar las raices de una ecuacin no lineal, siendo estos los valores de $x$ donde $f(x) = 0$

Para el mtodo de biseccin se obtiene la media de dos valores de x y se detendr hasta encontrar el intervalo donde la funcin cambia de signo. Este es el ms lento de los 3. Utiliza las iteraciones de la forma:

$c = \frac{1}{2}(a+b)$

En el mtodo de Newton se usa la tangente a la curva (la derivada) y se ir iterando hasta que el valor de $f(x)$ sea menor que el de la tolerancia $\epsilon$. Debido a que puede reducir en gran manera el intervalo, es el ms veloz de los 3 mtodos. Usa iteraciones de la forma:

$x_{k+1} = x_k - \frac{f(x_k)}{f'(x_k)}$

El mtodo de la secante, como su nombre lo indica, utiliza la secante para calcular las raices de la ecuacin, de igual manera, este iterar hasta que el valor de $f(x)$ sea menor que el de la tolerancia $\epsilon$. Este mtodo es ms veloz que el de la biseccin y un poco ms lento que el de Newton. Este utiliza las iteraciones de la forma:

$x_{k+1} = x_k - f(x_k)\frac{x_k - x_{k-1}}{f(x_k) - f(x_{k-1})}$

Inciso 2

Para implementar el mtodo de Newton en Python se crear la funcin correspondiente. Esta tomar los siguientes parmetros:

  • fx es la funcin a evaluar
  • dx es la derivada de la funcin
  • x0es el valor inicial $x_0$
  • e es la tolerancia
  • iter es el lmite de iteraciones a ejecutar.

La funcin tendr las excepciones para cuando se llega al mximo de iteraciones y cuando la derivada da 0, lo cual implica que no hay una solucin.

In[126]:
import numpy as np
import matplotlib.pyplot as plt
def Newton(fx, dx, x0, e, iter):
    x = x0
    k = 0
    for n in range(0, iter):
        fxn = fx(x)
        if abs(fxn) < e:
            print(f'Se lleg a la solucin x = {x} con {k} iteraciones')
            return x
        dfx = dx(x)
        if dfx == 0:
            print('Sin solucin, la derivada es 0')
            return None
        x = x - fxn/dfx
        k += 1
    print('Se lleg al mximo de iteraciones y no se encontr una respuesta')
    return None

def secante(f, x0, x1, e, iter):
    k = 0
    for n in range(iter):
        x2 = x1 - f(x1) * (x1 - x0) / (f(x1) - f(x0))
        if abs(f(x2)) < e:
            print(f"Se lleg a la solucin x = {x2} con {k} iteraciones")
            return x2
        x0, x1 = x1, x2
        k += 1

    print("Se alcanz el nmero mximo de iteraciones sin encontrar la raz.")
    return None
