{"categories": "[ \"MOBILE\" ]", "id": "d76f590d27f8eb415696e945f16b7e1c", "klass": "neutral", "text": "RT @AlbertoCiurana: Hay gente que se desconecta del mundo suicid\u00e1ndose. Yo solo apago el celular.", "type": "Celular"}

Los diccionarios contienen los campos:

  • categories

  • text
  • type

Van a usar el campo text para entrenar y la etiqueta ser el campo klass.

from io import StringIO

def read_json_lines(file_path):
    data = []
    with open(file_path, 'r') as file:
        for line in file:
            data.append(pd.read_json(StringIO(line), lines=True))
    return pd.concat(data, ignore_index=True)

#Se inicializa el lemmatizer
lemmatizer = nltk.WordNetLemmatizer()

# Definir funcin de preprocesamiento
def preprocess_text(text):
    # Normalizacin de texto para eliminar acentos
    text = unicodedata.normalize('NFKD', text).encode('ascii', 'ignore').decode('utf-8', 'ignore')
    # Convertir a minsculas
    text = text.lower()
    # Eliminar caracteres especiales
    text = re.sub(r'[^a-zA-Z\s@]', '', text)
    # Tokenizar el texto
    tokens = word_tokenize(text)
    # Eliminar stopwords, signos de puntuacin y nmeros y aplicar lemmatizacin
    tokens = [lemmatizer.lemmatize(token) for token in tokens if token.isalpha() and token not in stopwords.words('spanish')]
    # Unir tokens en una sola cadena
    return ' '.join(tokens)

from sklearn.metrics import accuracy_score
from sklearn.linear_model import LogisticRegression
from sklearn.svm import SVC

from sklearn.ensemble import RandomForestClassifier

# Definir una funcin para entrenar un clasificador
def train_classifier(model, X_train, y_train, X_val, y_val):
    model.fit(X_train, y_train)
    y_val_pred = model.predict(X_val)
    val_accuracy = accuracy_score(y_val, y_val_pred)
    return model, val_accuracy

Documentacin del mejor caso reportado

Preprocesamiento de Texto

Se prob quitar o no las stopwords, lematizar o no las palabras, as como no eliminar acentos y caracteres especiales, estos no generaron gran variacin, sin embargo los mejores resultados se obtuvieron con:

  • Conversin a minsculas: Todos los caracteres del texto se convierten a minsculas para asegurar la uniformidad.
  • Tokenizacin: El texto se divide en palabras individuales (tokens).
  • Eliminacin de stopwords: Se eliminan palabras comunes que no aportan mucho significado (e.g., "y", "el", "de").
  • Lematizacin: Se reducen las palabras a su forma base o lema (e.g., "corriendo" se convierte en "correr").

Vectorizador

Se prob todos los casos con los dos vectorizadores (CountVectorizer y TF-IDF), a pesar de que la variacin era pequea, en varios casos si se obtuvo algunas diferencias algo significativas, siendo el mejor caso:

  • TF-IDF (Term Frequency-Inverse Document Frequency): Convierte el texto en una matriz donde cada valor representa la importancia de una palabra en el documento y en el corpus.

Modelo de Clasificacin y Accuracy

Se probaron 4 diferentes modelos (Regresin Logstica, SVM, Naive Bayes y Random Forest), variando tambin los parmetros que estos aceptan, el mejor caso reportado es:
